== Installation ==

= Automatic Method =

1. Use WordPress `Dashboard -> Plugins -> Add New`
2. Search: `QuQuk`
3. Click `install now`
4. Activate the plugin `(in Plugins menu)`

= Manual Method =

1. Download the zip file.
2. Expand contents of zip file.  It will create a directory: `QuQuk`
3. Upload contents of zip file to your plugin directory: `/wp-content/plugins/`
4. Activate plugin
